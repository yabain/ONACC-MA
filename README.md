## Development Setup


### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]

### Setting Up a Project

Install the Angular CLI globally:

```
npm install -g @angular/cli
```

Open the project directory:

```
cd [PROJECT DIRECTORY]
```

Packages installation:

```
npm i
```

Start project

```
ionic serve --l
```
